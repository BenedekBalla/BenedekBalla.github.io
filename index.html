Ongoing and Past Projects

Accelerating ML Algorithms with Nvidia GPUs

Status : ONGOING (May 2024 - Present)

At the time of writing (June 9th 2024 - having spending the whole day studying CUTLASS GEMM), I am 1 month into this project and I am absolutely loving it. Over the past weeks, I have learned about Nvidia's Volta, Ampere, and Hopper GPU architectures, how these architectures work in tandem with the CUDA programming model, and how PyTorch's TorchDynamo and TorchInductor utilize OpenAI's Triton to automatically optimize high-level ML code to run on GPUs via a set of hard-coded compiler passes.

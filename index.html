Title : Accelerating ML Algorithms with Nvidia GPUs

Timeline : May 2024 - Present

Supervisor : Maryam Mehri Dehnavi

At the time of writing (June 2024), I have so far learned about Nvidia's GPU architectures, how these architectures work in tandem with the CUDA programming model, and how PyTorch's TorchDynamo and TorchInductor utilize OpenAI's Triton to automatically optimize high-level ML code for GPUs via a set of hard-coded compiler passes.

What did I learn?

  1. C++.
  2. Write efficient GPU kernels in CUDA and Triton.
  3. Nvidia Volta, Ampere, and Hopper architectures.
  4. Nvidia CUTLASS GEMM implementation.
  5. Just-in-time compilers, and compiler passes (E.g. TritonGPU IR passes).
  6. MLIR and LLVM IR.
  7. GPU-optimized ML algorithms (E.g. FlashAttention).

Title : DDPMs for Synthetic MSK-US Dataset Enrichment

Timeline : September 2023 - May 2024

Supervisor : Pascal Tyrrell

Over 8 months, I set out to improve the performance of a fine-tuned EfficientNet model at diagnosing thickened synovium and recess distension in the knee joint space. To accomplish this, I decided to enrich its tuning set using synthetically generated MSK-US images from a DDPM. The fact that I did not have access to a large amount of existing MSK-US images made it significantly harder to train my diffisuion model. To evaluate my results and methodology, I compared EfficientNet's performance when trained on a dataset enriched with synthetic images against enrichment with augmented images, generated with geometric and intensity transformations. My findings suggest that synthetic dataset enrichment is prefereable in the domain of medical imaging, since the diagnostic model's sensitivity increased by 4%, and the consistency of synthetic images with real images encouraged the model to learn representations aligned with human decision-making heuristics.

What did I learn?

  1. Build ML models in Python with PyTorch, PyTorch Lightning, Matplotlib, OpenCV, Numpy.
  2. Schedule and execute ML workloads on HPC clusters (Digital Research Alliance of Canada clusters).
  3. Write a publishable research paper (link coming soon!), and reading state-of-the-art ML research papers.
  4. The mathetmatical theory and Python implementation of Denoising Diffusion Probabilistic Models.
  5. Shell Script.

Title : Real-Time Gate Detection for Autonomous Drone Racing

Timeline : May 2023 - September 2023

Supervisor : Chao Qin

Joined a team of undergraduate, masters, and PhD students to build a competitive autonomous racing drone. I was tasked with finding a deep-learning algorithm capable of detecting drone racing gates, real-time, on the limited compute of the onboard hardware. My final solution successfully applied a shallow convolutional neural network and some smart filtering to improve inference time from over 300ms to less than 30ms.

What did I learn?

  1. State-of-the-art learning-based computer vision algorithms.
  2. Code in the Robotics Operating System.
  3. Sim-to-real transfer learning for training neural networks.

Projects

Title : Accelerating ML Algorithms with Nvidia GPUs

Timeline : May 2024 - Present

Supervisor : Maryam Mehri Dehnavi

At the time of writing (June 9th 2024 - having spent the day studying CUTLASS GEMM), I am 1 month into this project and I am absolutely loving it. Over the past weeks, I have learned about Nvidia's Volta, Ampere, and Hopper GPU architectures, how these architectures work in tandem with the CUDA programming model, and how PyTorch's TorchDynamo and TorchInductor utilize OpenAI's Triton to automatically optimize high-level ML code to run faster on GPUs via a set of hard-coded compiler passes.

What did I learn?

  1. C++.
  2. Write efficient GPU kernels in CUDA and Triton.
  3. Nvidia GPU architectures of V100, A100, H100.
  4. CUTLASS GEMM.
  5. Just-in-time compilers, and compiler passes (E.g. TritonGPU IR passes).
  6. MLIR and LLVM IR.
  7. GPU-optimized ML algorithms (E.g. FlashAttention).
  8. Understand professional Github repos (E.g. Triton, CUTLASS).

Title : DDPMs for Synthetic MSK-US Dataset Enrichment

Timeline : September 2023 - May 2024

Supervisor : Pascal Tyrrell

Over the course of 8 months, I set out to improve the performance of a fine-tuned EfficientNet model at diagnosing thickened synovium and recess distension in the knee joint space. To do so, I decided to enrich its tuning set using of MSK-US images using synthetic images generated by a DDPM. To evaluate my methodology, I compared the results achieved with synthetic images to simply using augmented images, generated using most geometric transformations. My findings suggest that synthetic dataset enrichment boosted the diagnostic model's sensitivity by 4%, outperforming augmented enrichment!

What did I learn?

  1. ML code with PyTorch, PyTorch Lightning, Matplotlib, OpenCV, Numpy.
  2. Schedule and execute ML workloads on HPC clusters (Digital Research Alliance of Canada) with state-of-the-art Nvidia GPUs.
  3. Writing my own research paper (currently deciding whether to publish it or not...), and reading others' research papers.
  4. Denoising Diffusion Probabilistic Model mathematical theory and Python implementation.
  5. Shell Script.
